# Jaya Ganguly - Professional Portfolio

A modern, animated portfolio website built with React, Vite, Framer Motion, and Tailwind CSS.

## Features

✨ **Stunning Animations**
- Smooth Framer Motion animations throughout
- Interactive hover effects
- Scroll-triggered animations
- Floating elements and transitions

🌓 **Dark/Light Theme Toggle**
- Seamless theme switching in navbar
- Persistent theme across sections
- Beautiful gradient accents

📱 **Fully Responsive**
- Mobile-first design
- Responsive navigation
- Optimized for all devices

🚀 **Performance Optimized**
- Built with Vite for fast loading
- Lazy loading animations
- Optimized images and assets

## Sections

1. **Hero** - Eye-catching introduction with floating cards
2. **About** - Professional summary with statistics
3. **Experience** - Timeline of professional growth
4. **Projects** - Featured projects with descriptions and links
5. **Skills** - Animated skill bars and competencies
6. **Contact** - Contact form and social links
7. **Footer** - Quick navigation and scroll-to-top

## Installation

1. Clone the repository
```bash
git clone https://github.com/Pseudocoder000/My_Portfolio.git
cd My_Portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

```bash
npm run build
```

The build output will be in the `dist/` directory.

## Preview Production Build

```bash
npm run preview
```

## Customization

### Change Colors
Edit the Tailwind config in `tailwind.config.js` to change the accent color and theme.

### Update Content
- **Hero**: Edit `src/components/Hero.jsx`
- **Projects**: Update project data in `src/components/Projects.jsx`
- **Skills**: Modify skill categories in `src/components/Skills.jsx`
- **Contact**: Update contact info in `src/components/Contact.jsx`

### Add Your Photo
Replace the placeholder image in `Hero.jsx` with your actual photo:
```jsx
src="path/to/your/photo.jpg"
```

## Deployment

### Vercel (Recommended)
1. Push code to GitHub
2. Connect repository to Vercel
3. Deploy with one click

### Netlify
1. Connect GitHub repository
2. Set build command: `npm run build`
3. Set publish directory: `dist`
4. Deploy
